# AIワークショップ リハーサル用教材

> **このリポジトリは社内向け Private リポジトリです。外部共有はご遠慮ください。**

---

## このリポジトリについて

社内AI活用ワークショップ（全25ユースケース構想）の**リハーサル用教材**を格納しています。
現在、3つのユースケース（#1〜#3）の教材が揃っており、テスター各位に実際の演習を体験していただき、本番展開に向けたフィードバックを集める段階です。

Claude Code Web + GitHub を使って「過去資料3件からパターンを抽出し、新規案件に適用する」体験ができます。各演習は**45分完結**を目標に設計しています。

---

## テスターの方へのお願い

以下の観点でフィードバックをください:

- [ ] **演習全体の所要時間**（ストップウォッチで測って教えてください）
- [ ] **詰まった箇所・分かりにくかった説明**（スクリーンショットがあれば尚可）
- [ ] **AIの出力が想定と違った場合**、その内容
- [ ] **timeout等のエラーが発生した場合**、頻度と対処
- [ ] **手順書の改善提案**
- [ ] **UIの説明で「どのボタンか迷った箇所」**があれば具体的に

**おすすめのテスト順序**: `#1 → #2 → #3` の順（難易度順。#1が最も素直に流れる設計、#3はDCF特有の制約があり難度最高）

---

## 各ユースケースの進め方（共通）

1. 対象のユースケースフォルダ（`usecase-01/` 等）を開く
2. フォルダ内の `README.md` を読む（各ユースケースの詳細案内）
3. `手順書_ClaudeCode版_XX_○○.docx` をダウンロードして開く
4. 手順書の指示に従って進める:
   - `ダミー_過去A/B/C` の3ファイルを Step1 で使用
   - `ダミー_新規` を Step2 で使用
5. 演習完了後、`模範_Step1_パターン定義` / `模範_Step2_○○` と自分の出力を見比べて到達度を確認
