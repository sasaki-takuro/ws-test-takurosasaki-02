過去3件（フェニックス商事・シグナス流通・ドラコマート）から抽出した共通ロジック。

## 実行ルール

### 作業スコープ
- **対象フォルダ**: `output/03_バリュエーションDCF自動構築/` のみ
- `output/01_*/`・`output/02_*/` 等の他ユースケースフォルダへの読み書き禁止

### 出力分割ルール
- 1ファイル **200行以内**・1セクション1ターン厳守
- 各セクションファイル保存後に必ず `git add / commit / push` を実行
- コミット後「セクション N 完了」とのみ報告し、「次」の指示まで待機

### ファイル命名規則
- 各セクション: `DCFモデル_<対象名>_<2桁連番>_<セクション名>.md`
  - 例: `DCFモデル_アルビレオ小売_01_前提条件.md`
- 統合版: `DCFモデル_<対象名>_人間向け.md`

### 統合版ルール
- 既存セクションファイルの連結（`cat`コマンド等）のみ
- 新規本文生成・内容改変・見出し追加は禁止

### エラー時の挙動（Stream idle timeout 発生時）
1. GitHubで当該ブランチの実ファイルを確認し、どこまで保存済みか把握する
2. 新セッションで未完了セクションから再開する
3. 重複コミットしないよう、git logで最終コミットを確認してから再開

### Python禁止
- DCF計算・感度分析も **Markdownテーブルに直接記述**すること
- `.py`・`.ipynb` ファイルは一切作成しない

| 予測期間 | **5期**（標準）/ 3期（叩き台） | — | 設計判断 |
| 永久成長率 g | **1.0%**（保守的）/ 0.5〜1.5%の幅 | マクロ見通し | 設計判断 |

### 1-B. 情報粒度定義（必須付記）

各パラメータに以下のいずれかを付記し、**AIによる推測埋め禁止**:

| ラベル | 意味 |
|--------|------|
| **確定値** | 監査済み財務諸表・公式資料に基づく |
| **概算値・口頭** | 先方ヒアリング・担当者発言ベース（要書面確認） |
| **業界平均仮置き** | 類似業態平均値で代替（精度低・要精査） |
| **未確認** | 資料請求中または先方未提示 |
| **情報なし** | 入手不能・推計不可 |

> **禁止**: 情報粒度が「未確認」「情報なし」の項目をAIが勝手に埋めること。
> 空欄または「未確認」と明記し、入手後に担当者が更新する。

## カテゴリ 2: PL予測テンプレート（5期）

### 算出式と整合条件

```
売上高(t) = 売上高(t-1) × (1 + 既存店成長率) + 新店寄与売上
売上総利益 = 売上高 × (1 - 売上原価率)
営業利益 = 売上総利益 - 販管費（人件費＋賃料＋販促費＋その他）- 減価償却費
EBITDA = 営業利益 + 減価償却費
経常利益 = 営業利益 - 支払利息
当期純利益 = 経常利益 × (1 - 実効税率)
```

### 整合条件（必須チェック）

| チェック項目 | 条件 |
|------------|------|
| 営業利益率 | 0%以上（赤字は別途要因分析） |
| 粗利率トレンド | PB比率向上・仕入改善なければ固定または微増にとどめる |
| 販管費率 | 最低時給上昇を人件費に織り込む（年+3%が標準仮定） |
| EBITDA マージン | 業態標準比較で乖離±3%以内か確認 |

## カテゴリ 3: CF予測テンプレート（5期）

### FCF算出式（標準型・BS連動）

```
NOPAT = 営業利益 × (1 - 実効税率)
営業CF = NOPAT + 減価償却費 - 運転資本増加額
  運転資本増加額 = (売上債権+棚卸資産-仕入債務)(t) - (売上債権+棚卸資産-仕入債務)(t-1)
投資CF = -CAPEX
FCF = 営業CF + 投資CF
```

> **アンチパターン（シグナス流通）**: BSなしで運転資本増減を無視するとFCFが過大計上になる。
> 食品SM等の棚卸資産が多い業態では特に歪みが大きい。必ずBS連動で算出すること。

### ターミナルバリュー（TV）算出ルール

```
TV = FCF(最終年) × (1 + g) / (WACC - g)
TV現在価値 = TV / (1 + WACC)^n   ※ n = 予測期間年数
```

条件: `WACC > g` であること（通常 g ≤ 1.5%、WACC ≥ 4%）

## カテゴリ 4: WACC算定テンプレート

### 算定式（CAPM＋加重平均）

```
Re（株主資本コスト） = Rf + β × MRP
WACC = Re × (E/V) + Rd × (1 - t) × (D/V)
  E/V = 株主資本 / (株主資本 + 有利子負債)
  D/V = 有利子負債 / (株主資本 + 有利子負債)
```

### パラメータ標準値（フェニックス商事リファレンス）

| パラメータ | 標準値 | 情報粒度 | 根拠 |
|-----------|--------|---------|------|
| 無リスク金利 Rf | **0.9%** | 確定値 | 10年国債利回り |
| マーケットリスクプレミアム MRP | **6.0%** | 確定値 | 過去20年平均 |
| レバードβ | **0.95** | 概算値 | 類似上場5社中央値 |
| 自己資本比率 E/V | 案件による（参考58%） | 確定値 or 概算値 | BS実績 |
| 負債コスト Rd（税前） | **1.8%** | 確定値 | BBB相当格付 |
| 実効税率 t | **30.0%** | 確定値 | 現行制度 |
| **WACC（算出値）** | **4.8%**（参考） | 算出値 | 上記から計算 |

> **アンチパターン（シグナス流通）**: WACCを根拠なく単純固定（例: 5.5%）するのは不採用理由になる。
> 必ず Rf・MRP・β・D/V比率から積み上げ計算すること。
