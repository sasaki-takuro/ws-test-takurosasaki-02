# DCFモデル 03 — CF予測・FCF算出（アルビレオ小売・3年）

テンプレート参照: `CLAUDE.md` § カテゴリ3　単位: 百万円

---

## 算出式

```
NOPAT         = 営業利益 × (1 − 実効税率30%)
営業CF        = NOPAT + 減価償却費 − 運転資本増加額
投資CF        = −CAPEX（新店＋既存改装＋維持修繕）
FCF           = 営業CF + 投資CF
ターミナルTV  = FCF(Y3) × (1 + g) / (WACC − g)
```

> **BS連動不可の旨**: 先方依頼により簡易版。運転資本増加額はBS実績値によらず
> 売上増加額の約1.5%（概算）で推計。正式評価時は担当者がBS連動値に手動補正すること。

## ターミナルバリュー（算出ルール）

```
TV = FCF(Y3) × (1 + g) / (WACC − g)
   = 6,993 × 1.010 / (WACC − 1.0%)
```

WACCは§04で確定。暫定WACC≈4.2%（§04参照）での参考計算:

```
TV（暫定）= 6,993 × 1.010 / (4.2% − 1.0%)
           = 7,063 / 0.032
           = 220,719 百万円
```
