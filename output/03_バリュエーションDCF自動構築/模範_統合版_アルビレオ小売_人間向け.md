## 3. CF予測(セクション03)

### 計算ロジックの説明

フリーキャッシュフロー(FCF)を以下の式で算出:

```
FCF = NOPAT + 減価償却費 - 運転資本増加額 - 設備投資
NOPAT = 営業利益 × (1 - 法人税率)
```

### 主要項目の前提

| 項目 | 値 | 根拠 |
|---|---|---|
| 法人税率 | 30.5% | 過去3期実効税率平均 |
| 減価償却費 | 売上比1.5% | 過去A完備版から流用、出店ペースが同等 |
| 運転資本/売上 | 5.0%(横ばい) | 過去5年平均、流通業界の標準 |
| 設備投資 | 売上比3.0% | 既存店改装1.5% + 新規出店1.5% |

### FCF予測

| 指標 | 20XX+1 | 20XX+2 | 20XX+3 | 20XX+4 | 20XX+5 |
|---|---|---|---|---|---|
| 営業利益 | 65億円 | 67億円 | 70億円 | 72億円 | 73億円 |
| NOPAT(税引後) | 45億円 | 47億円 | 49億円 | 50億円 | 51億円 |
| (+) 減価償却費 | 8億円 | 8億円 | 9億円 | 9億円 | 9億円 |
| (-) 運転資本増加 | 1億円 | 1億円 | 1億円 | 1億円 | 1億円 |
| (-) 設備投資 | 15億円 | 16億円 | 16億円 | 17億円 | 17億円 |
| **FCF** | **37億円** | **38億円** | **41億円** | **41億円** | **42億円** |

### ターミナルバリューの算定

ゴードンモデル(永続成長モデル)で算定:

```
ターミナルバリュー = 20XX+5年のFCF × (1 + 永続成長率) ÷ (WACC - 永続成長率)
                  = 42億円 × (1 + 2.0%) ÷ (6.5% - 2.0%)
                  = 42.84億円 ÷ 4.5%
                  ≈ 952億円
```

## 4. WACC算定(セクション04)

### 算定方法

CAPM(Capital Asset Pricing Model)を用いて加重平均資本コスト(WACC)を算定する。

### 構成要素

| 要素 | 値 | 出典・根拠 |
|---|---|---|
| リスクフリーレート(Rf) | 0.8% | 10年国債利回り(評価時点) |
| 株式リスクプレミアム(MRP) | 6.0% | Ibbotson Associates 等の業界標準値 |
| ベータ(β) | 0.85 | 類似企業4社の平均(食品SM・生活雑貨複合) |
| 株主資本コスト(Re) | 5.9% | Re = Rf + β × MRP = 0.8% + 0.85 × 6.0% = 5.9% |
| 負債コスト(Rd) | 1.2% | 同社の主要借入金利(税引前) |
| 税率(T) | 30.5% | 法人実効税率 |
| 負債コスト(税引後) | 0.83% | Rd × (1-T) = 1.2% × 69.5% |
| 自己資本比率(E/V) | 68% | 同社の最適資本構成(既存実績ベース) |
| 負債比率(D/V) | 32% | 1 - E/V |

### WACC 計算

```
WACC = (E/V) × Re + (D/V) × Rd × (1-T)
     = 68% × 5.9% + 32% × 0.83%
     = 4.01% + 0.27%
     = 4.28%
```

しかし、この値は流通業界の標準的な範囲より低い。本評価では**業界比較ベースで補正**し、最終的に **WACC = 6.5%** を採用する。

### WACC補正の根拠

- 過去A(フェニックス商事)の DCF モデルで使用された WACC は7.0%
- 過去B(シグナス流通)はそもそも WACC の根拠が不明で、投資委員会で問題視された
- 業界平均(類似上場企業4社の WACC レンジ)は6.0〜7.5%
- 単純な CAPM 計算より、業界バンドレンジで6.5%(中位)を採用するのが妥当

### 重要な注意

WACC は本 DCF モデルにおいて**最も感度が高いパラメータ**である。本評価では基本ケースとして6.5%を採用するが、§5 感度分析で ±1% の振れ幅を確認する。投資判断会議では「WACC 6.5% の根拠」を議論の重点とすべき。

## 7. まとめ:投資判断会議への提出

### 推定企業価値レンジ

| ケース | EV |
|---|---|
| 楽観 | 1,250億円 |
| **基本** | **1,048億円** |
| 悲観 | 850億円 |

基本ケースで約1,000億円。仮に提示価格がこのレンジ内であれば投資妥当性の検討に値する。

### 投資判断会議への論点提示

1. **WACC 6.5% の妥当性**:CAPM計算では4.28%だが、業界補正で6.5%とした。投資委員会で本案件のリスクプレミアムをどう評価するかを議論したい。
2. **永続成長率2.0%**:基本ケースで採用。国内消費成長と業界実績ベースだが、競合圧力を踏まえると1.5%への引下げ余地あり。
3. **悲観シナリオでの投資妥当性**:EV 850億円でも投資 IRR が確保できる買収価格レンジを設定すべき。

### 過去事例から学ぶ留意点

- 過去B(シグナス流通)で問題視された3点(BS欠落・WACC根拠・感度1軸)は本評価では全てカバー済み
- 過去A(フェニックス商事)と同等以上の情報粒度で評価できている
- 投資判断会議では「数値結果」より「前提値の妥当性」を中心に議論する設計
